# 🚗 ESP32 Bluetooth RC Car

A Bluetooth-controlled 4WD robotic car built using the **ESP32 DevKit** and **L298N Motor Driver**. The car is controlled wirelessly using the **Arduino Bluetooth Controller** Android application.

## ✨ Features

- 📱 Bluetooth control using Android smartphone
- 🚗 Forward, Backward, Left & Right movement
- 🛑 Stop function
- ⚡ ESP32-based control system
- 🔋 Battery-powered 4WD robot
- 🔧 Easy to upgrade and customize

## 🚀 Getting Started

1. Install Arduino IDE.
2. Install the ESP32 board package.
3. Open the project in Arduino IDE.
4. Upload the code to the ESP32.
5. Pair your Android phone with the ESP32.
6. Open **Arduino Bluetooth Controller**.
7. Connect and control the robot.
